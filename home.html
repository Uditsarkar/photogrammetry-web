Classification

To classify the land cover, we use supervised classification which are maximum likelihood (MLC) and support vector machine (SVM).

Maximum Likelihood Classification (MLC)

MLC has assumption that data points (pixel value in our case) in each class is normally distributes and it aims to fit the normal distribution on the data of each class. From the above figure, the probability of x  belonging to class B is higher than class A.

Support Vector Machine (SVM)

SVM aims to find the optimal hyperplane to separate different classes and maximized the margin. If the data is non-linearly separable, we can use kernel functions to transform the data into higher-dimensional space. The example of of kernel functions are Linear, Polynomial, Sigmoid, and Radial Basis Function (RBF).

For our classification, we use Radial Basis Function (RBF) kernel Type.
